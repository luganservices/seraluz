# Seraluz — Velas Aromáticas Artesanais

Site institucional e vitrine de produtos da **Seraluz**, loja de velas aromáticas
artesanais feitas com cera de coco 100% vegetal, essências lipossolúveis e pavio
de algodão.

## Sobre o site

Site estático (HTML + CSS + JavaScript puro), sem dependências ou build.

### Seções
- **Hero** — apresentação da marca
- **Vitrine de produtos** — catálogo com preços e botão de pedido via WhatsApp
- **Nossa história** — diferenciais e infográfico de produção
- **Depoimentos** — avaliações de clientes
- **CTA** — links para Instagram e WhatsApp
- **Rodapé** — contato e política de troca e devolução

## Estrutura

```
seraluz/
```

## Como rodar localmente

Basta abrir o arquivo `index.html` em qualquer navegador.
